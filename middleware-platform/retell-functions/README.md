# Retell Functions

This folder contains Retell AI function configurations.

## Files

- `collect-insurance.json` - Insurance collection function configuration

## Quick Setup

1. Copy the JSON from `collect-insurance.json`
2. Go to Retell Dashboard → Functions → Add Function
3. Paste the JSON
4. Update `function_url` with your Railway domain
5. Save

For detailed setup instructions, see `docs/MASTER_DOCUMENTATION.md` section 5 (Voice Agent Integration).
